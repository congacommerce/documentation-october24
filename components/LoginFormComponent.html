Description

Login Form component is used to display login page where user can register or log into the application.

Preview

Usage

Example :
import { LoginFormModule } from '@congacommerce/elements';
@NgModule({
imports: [LoginFormModule, ...]
})
export class AppModule {}
Example :
// Basic usage.
```typescript
<apt-login-form
            [type]="'OAuth'"
            (onRegister)="handleOnRegisterChange()"
            (onLogin)="handleOnLoginChange()"></apt-login-form>
